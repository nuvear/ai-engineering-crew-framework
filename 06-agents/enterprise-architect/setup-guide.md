# Enterprise Architect Agent — Setup Guide

## Gemini Enterprise Setup

Follow these steps to configure the Enterprise Architect Agent:

```text
1. Open Gemini Enterprise.
2. Go to Agent Designer.
3. Create new single-step agent.
4. Name it Enterprise Architect Agent.
5. Paste the system instruction.
6. Connect framework knowledge documents.
7. Test using sample inventory management project.
8. Validate that the agent does not start coding.
```

## Knowledge Documents to Connect

- Framework README and operating model documents
- Agent topology and handover maps
- Approval gates overview
- Reference project (inventory management system)

## Validation

After setup, run the test plan in `test-plan.md` to confirm the agent behaves correctly.
