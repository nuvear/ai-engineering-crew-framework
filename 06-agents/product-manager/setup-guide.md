# Product Manager Agent — Setup Guide

## Gemini Enterprise Setup

1. Open Gemini Enterprise Agent Designer
2. Create new single-step agent named Product Manager Agent
3. Paste system instruction from `system-instruction.md`
4. Connect product plan template and reference project sample
5. Connect requirement specification template

## Activation Prerequisites

- Gate 1 (Requirement Approval) must be approved
- Approved Requirement Specification must exist in GitHub
- Requirement-to-Product Handover must be available
