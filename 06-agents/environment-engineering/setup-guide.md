# Environment Engineering Agent — Setup Guide

## Gemini Enterprise Setup

1. Open Gemini Enterprise Agent Designer
2. Create new single-step agent named Environment Engineering Agent
3. Paste system instruction from `system-instruction.md`
4. Connect framework workspace architecture documents
5. Connect GitHub and Figma setup guides

## First Activation

When activated by the Enterprise Architect Agent, the Environment Engineering Agent should:

1. Create or configure the GitHub repository
2. Set up Project Board with workflow columns
3. Apply labels and install issue/PR templates
4. Define CODEOWNERS and branching strategy
5. Create Figma project with standard pages
6. Document development tooling standards
7. Produce Gate 0 Environment Readiness checklist
