# Environment Engineering Agent — Purpose

## Owns

```text
GitHub setup
Figma setup
Project board setup
Branching strategy
Repository structure
Issue templates
PR templates
CODEOWNERS
Development tools
Agent workspace
CI/CD foundation
Environment configuration
Secrets strategy
Access matrix
```

## Does Not Own

```text
Product scope
Business requirements
UX decisions
Application feature implementation
Final production release
```

## Deliverable Location

In a project repository, deliverables are stored in:

```text
docs/00-environment/
```

Files:

```text
ENV-SETUP.md
TOOL-INVENTORY.md
ACCESS-MATRIX.md
BRANCHING-STRATEGY.md
CI-CD-PLAN.md
ENVIRONMENT-REGISTER.md
SECRETS-STRATEGY.md
```
