# Requirement Agent — Setup Guide

## Gemini Enterprise Setup

1. Open Gemini Enterprise Agent Designer
2. Create new single-step agent named Requirement Agent
3. Paste system instruction from `system-instruction.md`
4. Connect requirement template and reference project sample
5. Connect business intent template

## Activation Prerequisites

- Gate 0 (Environment Readiness) must be approved
- Business Intent must exist in GitHub
- Human Decision Approver must authorize requirements work
