# Requirement Agent — Test Plan

## Test Prompt

```text
Requirement Agent,

Project:
Inventory Management System

Goal:
Track products, stock quantity, stock movements, low-stock alerts, and simple reports.

Business owner and warehouse staff.
```

## Success Criteria

```text
Every requirement has ID
Every requirement links to business goal
Every requirement has acceptance criteria
Out-of-scope is clear
No architecture decisions
No code
Handover sections exist
```
