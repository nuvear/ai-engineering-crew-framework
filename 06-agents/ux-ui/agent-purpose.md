# UX/UI Agent — Purpose

## Two Modes

```text
Mode 1: Experience Design / UX
Mode 2: Visual Design / UI
```

UX must be approved before UI work begins.

## Owns

```text
Personas
User journeys
Task flows
Information architecture
Screen inventory
Wireframes
Clickable prototype
Sample data
User acceptance scenarios
Brand application
Visual design
Design system
Developer handoff
```

## Does Not Own

```text
Product scope
Architecture
Code implementation
Final business approval
```
